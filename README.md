## Título do Projeto
ANCI - Conectando Gerações 

## Descrição do Projeto
O app tem como objetivo ajudar idosos a usar melhor o smartphone por meio de uma interface simples, clara e totalmente pensada para acessibilidade.
O ANCI busca promover autonomia digital para pessoas idosas, oferecendo tutoriais passo a passo, lembretes importantes e uma navegação amigável. A ideia é reduzir a dificuldade que muitos idosos têm com tecnologia e facilitar tarefas básicas do dia a dia no celular.
